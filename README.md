# Housing Agent

This project is a housing search intake agent.

Goal:
automatically find new apartment listings that match predefined search criteria
and later send approved results into a Google Sheets tracker.

Current stage:
- local Python project created
- GitHub repository connected
- GitHub Actions workflow works
- search configuration module created
- search plan module created
- direct StreetEasy requests return 403
- next step is browser-based cloud collection

Current search target:
- source: StreetEasy
- track: apartment
- max price: 2750
- home types: studio, 1 bedroom
- neighborhoods: Astoria, Jamaica, Jackson Heights, Woodside, Ditmars-Steinway
